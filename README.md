# Ligando a Luz
Esse é um mini projeto que tem um interruptor que altera as cores de fundo. Resolvi usar um gradient no fundo e como ele não interage com o transition, usei JavaScript para tentar fazer uma transição de cores mais suave.
> Esse projeto usou de refêrencia o seguinte vídeo: [Botão CheckBox Customizado](https://youtu.be/DrHzKZh2FmM?si=WtO5P2uHkQrBQjC9)

## Tecnologias
- `HTML`
- `SCSS`
- `JavaScript`

## Características
- Ao pressionar o interruptor, a `Smiley-Face` gira para o outro lado.
- Quando o interruptor é ligado o fundo vai mudar para cores mais claras.

## Processo
Tinha encontrando um vídeo no youtube de alguém ensinando a fazer um interruptor animado e decidi fazer o meu. No vídeo foi usando o gradient de cores para personalizar a cor do fundo e fiz isso no meu projeto também, porém o transition não funcionava com o `radial-gradiente()` do css então me propus o desafio de fazer uma transição suave de cores com o JavaScript.
Inicialmente era para ser um interruptor que trocava do tema claro para o escuro, e foi dai que veio a ideia de ligar a luz.
